# 추상전략 게임들을 구현합니다!

python의 pygame을 이용하여 추상전략 게임을 구현해봅니다.

추상전략 게임이란 보드 게임에서 숨겨지거나 운적인 요소가 없고 나와 상대방의 패가 모두 공개된 게임을 말합니다. 수가 유한하기 때문에 서로 최선의 수를 두었을 때 한 쪽이 무조건 이기거나 무조건 비길 수 밖에 없습니다. 그래도 무궁무진한 수를 가지고 있기 때문에 충분히 즐길 수 있는 게임입니다.
